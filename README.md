# Apprentissage JavaScript

Projet personnel de reconversion vers le développement web front-end.

Ce dépôt regroupe mes exercices réalisés pendant mon apprentissage progressif du JavaScript, en parallèle de cours en ligne (OpenClassrooms) et de pratique personnelle.

## Objectif
Acquérir les bases nécessaires pour intégrer une formation diplômante puis exercer comme développeur junior.

## Méthode de travail
- Étude d’un concept
- Reproduction sans copier-coller
- Adaptation personnelle
- Validation par petits exercices

## Compétences en cours d’acquisition
- variables et types
- conditions
- boucles
- tableaux et objets
- manipulation du DOM
- événements utilisateur

## Organisation du dépôt
Chaque fichier correspond à une notion précise étudiée chronologiquement.

## Remarque
Ce dépôt montre ma progression d’apprentissage : le code n’a pas vocation à être parfait mais à illustrer la compréhension progressive des concepts.
